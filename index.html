My Services

POSTMAN

Postman is a computer application used for API testing. Postman sends an API request to a web server and receives a response, whatever that may be. No additional work or framework settings are required when sending and receiving requests in Postman. Widely used by Testers and Developers for better application testing. Easily integrates with your Continuous Integration (CI) & Continuous Development Pipeline.

KATALON

Katalon Studio is an open source application for automated testing developed by Katalon LLC. Katalon supports 3 automated testing platforms, namely Web testing, API testing, and Mobile testing.

CYPRESS

By using Cypress, users can perform several types of tests such as: end to end tests, integration tests, and unit tests. Cypress makes it easy for users to manage tests, write tests and run tests faster, easier, and more reliably.

JMETER

Java-based open source application that can be used for performance tests. For a QA Engineer, jMeter can be used to load/stress test Web Applications, FTP Applications and Database server tests.

Test Case

Making test cases aims to ensure that a system can run properly according to initial requirements and is able to respond when there is an invalid input. Test Case has several components such as test case id, test case description, precondition, test step, expected result, actual result, and status. Test Case acts as a starting point in the implementation of testing a system. From these test cases it is usually known whether the system features are running normally or not
